# Data Instructions

This project uses the PerSeg dataset released with the PerSAM paper and the SAM ViT-H checkpoint.

Due to file size, the dataset and checkpoint are not included in this repository.

## Required files

1. PerSeg dataset
2. SAM ViT-H checkpoint: `sam_vit_h_4b8939.pth`

## Expected directory structure

After downloading, organize the official PerSAM repository as:

```text
Personalize-SAM/
├── data/
│   ├── Images/
│   └── Annotations/
└── sam_vit_h_4b8939.pth
